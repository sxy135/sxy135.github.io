Project Introduction

Course Selecting Assistant is an APP which aims at helping students to select courses and check the timetable under any circumstances and at any time. It's a course project which required us to evaluate and redesign a specific system in our school. We chose to evaluate our course selecting system and develop an APP to provide more convenience to students.




Project information

  • Type :A group project for the Human Computer Interaction course.
  • Team Member :Xiaoyu Sun, Xiangyi Feng
  • My Role :Group Leader, UI/UX Design, Usability Evaluation, User Research
  • Date :Nov 2015 - Jan 2016


Usability Evaluation


We chose the course selecting system as our study theme because we all used it very frequently at the start of each semester, and this website did cause a lot of troubles to us due to its bad design.

To begin with, we used two popular usability evaluation methods which are cognitive walkthrough and heuristic evaluation to find out the usability flaws in the old course selection system.






User Research


We briefly interviewed ten students from different grades and different majors about their opinions about the current system. And we gathered the answers that are most representative.



Information Architecture


According to the information we gathered above, we used post-it notes to brainstorm and then concluded the primary user demands as below.


According to the feedback of our students, a mobile version of the current course selecting system is in high demand since the website is not responsible and students like to use mobile Apps to help them to do things easily at any time and any place.

Then we decided to design a mobile version of the current course selecting system to helps our students solve the problems they encountered. According to the user needs that we have analyzed above, we built information architecture of this APP. The functions are divided into four parts which are Selecting Courses, My Timetable, Personal Center and Setting.

The index is My Timetable because it is the most frequently checked information.
