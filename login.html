Project Overview

  • About : This is a redesign project of a complicated data analytics system of Nielsen called KMA.
    We managed to integrate the old KMA system into the new Nielsen services platform.

    We also fixed the usability issues of the current system to reduce the learning efforts and to improve the efficiency.
  • My Role : Lead UX Designer - participated in user research, product strategy, prototyping, and user testing
  • Timeline : Sep 2019 - Present, plan to launch around Sep 2020
  • Impact : Largely reduced training time, improved user satisfaction, and consistency with Nielsen design system
